What is User Agent?

User agent is not a person. User agent is a software (also called a software agent) that operates on behalf of a user, such as web browser the “retrieves, renders and encourages the interaction of end-users with the Web content

User Agent's purpose

As an application, it is used for one of the TCP / IP networking protocols — specifically, it is the one that links the website you are trying to access. "User agent" refers to a program that accesses another device remotely, normally a server, through the network

History of User Agent

  1. The first web browser, Mosaic, was released in 1993 by the National Center for Supercomputing Applications (NCSA). User agent it uses is NCSA_Mosaic/2.0. After that, Mozilla browser was invented (it has an originally named Netscape than renamed to Firefox). At first, Mozilla used a user agent Mozilla/1.0.
  2. After Netscape was presented, Microsoft also invented Internet Explorer and they hoped that it would be ‘Netscape Killer’. And as expected, Internet Explorer emerged, overtaking and beating the Netscape. But then Netscape was reborn as Mozilla. Mozilla developed Genko, using the user agent Mozilla/5.0
  3. Linux also develops the KHTML engine but is not well supported as Gecko, so they added the user agent Mozilla/5.0 (compatible; Konqueror / 3.2; FreeBSD) (KHTML, like Gecko) string and developed it to be compatible with frame of Gecko.
  4. Apple developed another web browser called Safari. They named their engine WebKit. Used Mozilla/5.0 (Macintosh; U; PPC Mac OS X; de-de) AppleWebKit/85.7 (KHTML, like Gecko) Safari/85.5
  5. After all, Google invented Chrome. Chrome used user agent Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US) AppleWebKit/525.13 (KHTML, like Gecko) Chrome/0.2.149.27 Safari/525.13.a
